من به یک افزونه وردپرسی که یک سیستم قوی و کامل و استاندارد و حرفه ای هست نیاز دارم که ثبت نام و ورود باهاش انجام بشه درسایت 
و فرم ورود و ثبت نام به جای فرم ورود و ثبت نام پیش فرض وردپرس پیاده بشه 
من از پنل پیامکی در سایت https://app.sms.ir/ استفاده میکنم 
لطفا ابتدا این سایت و مستندات api و غیره رو بررسی کن و بعد اقدام به طراحی افزونه بکن 
یک افزونه استاندارد و ایمن و حرفه ای با ظاهر گیمینگ یک سیستم کامل ورود و ثبت نام با کد تایید
لطفا برای ست کردن api key و کد شناسه قالب و غیره رو در داشبورد وردپرسی بزار تا از اونجا بشه تنظیمات رو ست کرد
لطفا افزونه رو کاملا بهینه طراحی کن که کدها در فایل های خود باشن
لطفا به مرحله production-ready , standar, پالت رنگی و استایل گیمینگ.


## سناریوی ورود/ثبت‌نام
1. کاربر شماره موبایل وارد می‌کند.
2. کد تایید پیامکی ارسال می‌شود.
3. اگر کاربر قبلاً وجود داشته باشد، با کد درست مستقیم وارد سایت می‌شود.
4. اگر کاربر جدید باشد، بعد از تایید کد فیلد یوزرنیم نمایش داده می‌شود.
5. یوزرنیم به‌صورت زنده بررسی می‌شود:
   - اگر تکراری باشد خطا نشان می‌دهد.
   - اگر آزاد باشد پیام موفق با آیکن ✅ نمایش می‌دهد.
6. ثبت‌نام تکمیل و کاربر لاگین می‌شود.
